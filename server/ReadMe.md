1. **安装flask（如有必要）**

   ```python
   pip intall flask
   ```

   

2. **安装flask_cors（若有必要）**

   ``` python
   pip install flask_cors
   ```

   

3. **运行服务器app**

   ``` py
   python server.py
   ```

   

4. 目录结构：

   * templates
     * index.html：测试上传图片用
   * upload：上传的图片会被重命名并保存在此文件夹中
   * server.py：server api文件，可运行
   * strUtil.py：辅助文件
